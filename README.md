# Welcome to my website
A simple personal website built with Flask to showcase my projects, experience, and web development skills.

This site serves as both an online profile and an example of a full-stack project I built from scratch.

The website is hosted on pythonanywhere.com

LINK: http://origino.pythonanywhere.com/

## Features
- Custom personal webpage
- Backend built with Flask (Python)
- Clean HTML/CSS front-end
- Project sections, images, and personal details
- Fully customizable layout for future updates
- user contact database

## Tech used
### Front end
1. HTML
2. CSS

### Backend
1. Python
2. Flask
3. Routing

## How to run locally
1. clone https://github.com/chanchunkiu/profile
2. cd profile
3. python -m venv venv(virtual environment)
4. source venv/bin/activate
5. pip install -r requirements.txt
6. python server.py
7. Open browser http://127.0.0.1:5000
